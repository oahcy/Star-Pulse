# Star Pulse

## 项目简介

`Star Pulse` 是一个使用 Java + Swing 开发的 2D 俯视角小游戏工程，使用 Gradle 构建和运行。

当前版本已经实现了完整可玩的基础内容，包括：

- 玩家移动与朝向变化
- 自动索敌射击
- 普通敌人与 Boss
- 波次推进与计分
- 暂停、开始界面、重开
- 技能系统
- 掉落回血道具
- Boss 弹幕
- 图片与音效资源加载

## 快速开始

如果你只想快速体验游戏，在项目根目录执行：

```bat
.\gradlew.bat run
```

或：

```bash
./gradlew run
```

## 运行方式

推荐使用项目自带的 Gradle Wrapper。

构建项目：

```bash
./gradlew build
```

Windows 下：

```bat
.\gradlew.bat build
```

运行游戏：

```bash
./gradlew run
```

Windows 下：

```bat
.\gradlew.bat run
```

## 操作说明

- `W/A/S/D` 或方向键：控制玩家移动
- `Space`：释放脉冲技能
- `P` 或 `Esc`：暂停 / 继续
- `Enter`：开始游戏、暂停恢复、死亡后重开
- 鼠标：点击开始界面和结算界面的按钮

## 当前玩法

### 基础战斗

- 玩家出生在地图中央
- 敌人会随机生成并追踪玩家
- 当敌人进入一定范围后，玩家会自动发射子弹攻击
- 玩家会根据移动方向改变朝向，并保持最后一次移动方向

### 敌人与波次

- 普通波次会持续刷新普通敌人
- 每隔 3 波会出现一个 Boss 波次
- Boss 体型更大、生命更高、伤害更高
- Boss 会额外发射远程弹幕

### 技能与掉落

- 玩家拥有脉冲技能，释放后会：
  - 清除当前场上的敌方子弹
  - 对周围敌人造成范围伤害
- 敌人死亡后有概率掉落回血道具
- 玩家拾取道具后会恢复生命值

### 界面与反馈

- 游戏包含开始界面、暂停界面和结算界面
- 开始界面和结算界面支持鼠标按钮点击
- 左上角显示：
  - 玩家血量
  - 当前波次
  - 当前得分
  - 当前敌人数
  - 技能冷却状态
- 玩家与敌人受击时会有闪烁反馈
- 使用了真实的 PNG 图片资源和 WAV 音效资源

## 项目结构

主要代码位于 `src/main/java/aicode`：

- `Game.java`
  - 游戏窗口、主循环、输入处理、界面状态切换

- `GameMap.java`
  - 地图数据、波次逻辑、对象更新、HUD 绘制

- `Spriters.java`
  - 基础精灵类，提供位置、碰撞、受伤、绘制接口等通用能力

- `Player.java`
  - 玩家逻辑，包括移动、朝向、自动攻击、技能冷却、回血

- `Enemy.java`
  - 普通敌人逻辑

- `BossEnemy.java`
  - Boss 敌人逻辑，包含更强属性和远程弹幕

- `Bullet.java`
  - 玩家子弹逻辑

- `EnemyBullet.java`
  - 敌方子弹逻辑

- `Pickup.java`
  - 掉落物逻辑

- `SpriteAssets.java`
  - 图片资源加载

- `AudioAssets.java`
  - 音频资源加载

- `TonePlayer.java`
  - 音频加载失败时的回退合成音

整体职责可以简单理解为：

- `Game` 负责窗口、输入和游戏状态
- `GameMap` 负责地图中的对象管理与战斗流程
- `Spriters` 及其子类负责具体实体行为

## 资源目录

资源文件位于 `src/main/resources`：

- `images/`
  - 玩家、敌人、Boss、子弹、掉落物、背景贴图

- `audio/`
  - 射击、命中、Boss、波次、技能、回血、死亡等音效

如果后续想替换美术或音效，优先修改这个目录下的资源文件即可。

## 资源生成

项目中包含一个简单的资源生成器：

- `assets-gen/AssetGenerator.java`

它可以生成当前项目使用的基础 PNG 图片和 WAV 音效资源，方便后续继续替换或扩展素材。
